# Smart Temperature-Activated Fan System

An embedded Raspberry Pi automation system that dynamically controls fan oscillation speed and LED brightness based on environmental temperature and human presence detection.

The system integrates a thermistor, ultrasonic sensor, servo motor, and PWM-controlled LED using GPIO and I2C communication protocols.

## Demo Video

[Watch the System Demo](https://youtu.be/AmOVTb6TK0s)

## Features

- Real-time temperature monitoring using a thermistor
- Human presence detection using an ultrasonic sensor
- Adaptive fan oscillation speed control
- PWM-based LED brightness adjustment
- Multi-threaded servo motor control
- I2C communication with PCF8591 ADC/DAC module
